TL431 Compensation

PNG image placeholder Upload as images/IMG000_TL431_CV_Loop_01.JPG

Overview

This note derives the small-signal transfer function of a TL431-based constant-voltage feedback loop with Type-II compensation. The analysis starts from the output perturbation and models the TL431, optocoupler, and compensation network to obtain the loop transfer function. The resulting equation reveals one compensation zero, one pole at the origin, and one high-frequency pole.

Assumptions

  • VCC and VREF are treated as AC ground.
  • The TL431 REF pin is approximated as a virtual ground.
  • Rb provides DC bias only and is ignored in the AC model.
  • The optocoupler is modeled using a constant current transfer ratio (CTR).

Key Equations

i1 = Vo / RH

Z2 = R2 + 1/(sC2)

Vi = -i1·Z2

i2 = (Vo - Vi)/RLED

Zc = Rc || 1/(sCc)
   = Rc/(1 + sCcRc)

Vc = -CTR·i2·Zc

Transfer Function

          -CTR·Rc       1 + sC2(RH + R2)
Vc/Vo = ------------- · ------------------
        C2·RH·RLED      s(1 + sCcRc)

Poles and Zero

Zero:
fz = 1/[2πC2(RH + R2)]

Pole 1:
fp1 = 0 Hz   (integrator)

Pole 2:
fp2 = 1/(2πCcRc)

Summary

  • The network forms a Type-II compensator.
  • C2 and RH + R2 set the compensation zero.
  • The origin pole provides integral action and high DC loop gain.
  • Rc and Cc set the high-frequency pole.
  • CTR changes the loop gain but does not move the pole-zero locations.

I. Constant-Voltage Loop Analysis

Overview

This page replaces the TL431 symbol with an equivalent high-gain amplifier model. Because the non-inverting input is connected to the AC-grounded reference voltage, the feedback node is approximated as a virtual ground. The resulting small-signal derivation is therefore identical to the TL431 model used on Page 1.

Assumptions

  • VCC2 and VREF are treated as AC ground.
  • The amplifier input current is approximated as zero.
  • The feedback node is a virtual ground under high loop gain.
  • Rb provides DC bias only and is omitted from the AC model.
  • The optocoupler CTR is treated as constant.

Key Equations

i1 = Vo / RH

Z2 = R2 + 1/(sC2)

Vi = -i1·Z2

i2 = (Vo - Vi)/RLED

Zc = Rc || 1/(sCc)
   = Rc/(1 + sCcRc)

Vc = -CTR·i2·Zc

Transfer Function

          -CTR·Rc       1 + sC2(RH + R2)
Vc/Vo = ------------- · ------------------
        C2·RH·RLED      s(1 + sCcRc)

Poles and Zero

Zero:
fz = 1/[2πC2(RH + R2)]

Pole 1:
fp1 = 0 Hz   (integrator)

Pole 2:
fp2 = 1/(2πCcRc)

Summary

  • The equivalent amplifier model produces the same transfer function as the TL431 model.
  • The virtual-ground approximation simplifies the compensation-network analysis.
  • The pole-zero locations are unchanged from Page 1.
  • The model is useful when interpreting the TL431 as an error amplifier.

II. Constant-Current Loop Analysis

Overview

This page derives the small-signal transfer function of the constant-current regulation loop. The analysis begins with the current-sense voltage, which is amplified by gain K and applied to the Type-II compensation network. The optocoupler then transfers the compensation current to the isolated side.

Assumptions

  • The current-sense signal is represented by VM = K·ILOAD·Rsen.
  • VCC1, VCC2, and VIREF are treated as AC ground.
  • The amplifier input current is approximated as zero.
  • The feedback node is treated as a virtual ground.
  • Rb provides DC bias only and is omitted from the AC model.
  • The optocoupler CTR is treated as constant.

Key Equations

VM = K·ILOAD·Rsen

i1 = VM / R1

Z2 = R2 + 1/(sC2)

Vi = -i1·Z2

i2 = -Vi/RLED

i3 = CTR·i2

Zc = Rc || 1/(sCc)
   = Rc/(1 + sCcRc)

Vc = -i3·Zc

Transfer Function

          CTR·Rc       1 + sC2R2
Vc/VM = ------------ · -------------
        C2·R1·RLED    s(1 + sCcRc)

Poles and Zero

Zero:
fz = 1/(2πC2R2)

Pole 1:
fp1 = 0 Hz   (integrator)

Pole 2:
fp2 = 1/(2πCcRc)

Summary

  • The constant-current loop also forms a Type-II compensator.
  • R2 and C2 set the compensation zero.
  • The origin pole provides integral action and high DC loop gain.
  • Rc and Cc set the high-frequency pole.
  • The sense-amplifier gain K and Rsen scale the complete current-loop gain.
  • CTR changes the loop gain without moving the compensation pole or zero.
